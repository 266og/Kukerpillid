HTML Tags Reference

  • <html>: The root element of an HTML page. Example: <html>...</html>
  • <head>: Contains meta-information about the document. Example: <head>...</head>
  • <title>: Sets the title of the document, shown in the browser tab. Example: <title>My Page</title>
  • <body>: Contains the content of the document. Example: <body>...</body>
  • <p>: Defines a paragraph. Example: <p>This is a paragraph.</p>
  • <a>: Defines a hyperlink. Example: <a href="https://www.example.com" target="_blank">Visit Example</a>
    Visit Example
  • <div>: A container for HTML elements. Example: <div>Content here</div>
  • <span>: A container for inline elements. Example: <span>Inline text</span>
  • <img>: Embeds an image in the document. Example: <img src="image.jpg" alt="Description">
  • <ul>: Defines an unordered list. Example: <ul><li>Item</li></ul>
  • <ol>: Defines an ordered list. Example: <ol><li>First item</li></ol>
  • <li>: Defines a list item. Example: <li>List item</li>
  • <h1>: Defines a heading level 1. Example: <h1>Main Heading</h1>
  • <h2>: Defines a heading level 2. Example: <h2>Subheading</h2>
  • <br>: Defines a line break. Example: <br>
  • <header>: Represents introductory content. Example: <header>...</header>
  • <nav>: Represents a section of navigation links. Example: <nav>...</nav>
  • <footer>: Represents the footer of a document. Example: <footer>...</footer>
  • <section>: Represents a thematic grouping of content. Example: <section>...</section>
  • <article>: Represents a self-contained composition. Example: <article>...</article>
  • <aside>: Represents content related to the main content. Example: <aside>...</aside>
  • <form>: Represents a form for user input. Example: <form>...</form>
  • <input>: Represents an input control. Example: <input type="text">
  • <button>: Represents a clickable button. Example: <button>...</button>
  • <table>: Represents tabular data. Example: <table>...</table>
    Name Age
    John Doe 30
  • <blockquote>: Represents a quoted section. Example: <blockquote>...</blockquote>
    This is a famous quote.
  • <label>: Represents a caption for an item. Example: <label>...</label>
  • <textarea>: Represents a multi-line text input. Example: <textarea>...</textarea>
  • <select>: Represents a menu of options. Example: <select>...</select>
  • <progress>: Represents the progress of a task. Example: <progress>...</progress>
    70%
  • <meter>: Represents a scalar measurement. Example: <meter>...</meter>
    60%
  • <canvas>: Used for drawing graphics. Example: <canvas>...</canvas>
  • <svg>: Represents Scalable Vector Graphics. Example: <svg>...</svg>
  • <iframe>: Embeds another HTML page. Example: <iframe>...</iframe>
  • <details>: Represents a disclosure widget. Example: <details>...</details>
    Click to expand

    This is additional information that can be toggled.

  • <time>: Represents a specific period in time. Example: <time>...</time>
    October 1, 2023
  • <abbr>: Represents an abbreviation. Example: <abbr>...</abbr>
    HTML
  • <cite>: Represents the title of a work. Example: <cite>...</cite>
    The Great Gatsby
  • <code>: Represents a fragment of code. Example: <code>...</code>
    console.log('Hello, World!');
  • <strong>: Represents strong importance. Example: <strong>...</strong>
    This text is important.
  • <em>: Represents emphasized text. Example: <em>...</em>
    This text is emphasized.
  • <small>: Represents small print. Example: <small>...</small>
    This is small text.
  • <mark>: Represents highlighted text. Example: <mark>...</mark>
    This text is highlighted.
  • <del>: Represents deleted text. Example: <del>...</del>
    This text has been deleted.
  • <ins>: Represents inserted text. Example: <ins>...</ins>
    This text has been inserted.
  • <sub>: Represents subscript text. Example: <sub>...</sub>
    H2O
  • <sup>: Represents superscript text. Example: <sup>...</sup>
    E = mc2
  • <address>: Represents contact information for the author or owner of a document. Example: <address>...</address>
    Contact us at: info@example.com
  • <template>: Represents a container for content that is not rendered when the page loads but can be instantiated later. Example: <template>...</template>
